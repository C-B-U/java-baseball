## 기능 명세
1. 랜덤 정답 생성 기능
    - 정답 저장 기능
2. 사용자 정답 입력 기능
    - input 숫자 여부 검증 및 예외처리
    - input 숫자의 길이 검증 및 예외처리
3. Strike와 Ball 개수 판단 기능
   - 3 Strike인 경우 게임 종료 기능
   - 3 Strike가 아닌 경우 결과 출력 기능
4. 정답인 경우 새 게임 진행 여부 입력 기능
   - input 1 or 2 검증 및 예외처리
   - input 1인 경우 새 게임 시작 기능
   - input 0인 경우 종료 기능
